# rel4-kernel-autobuild
本仓库用于集成式的构建[rel4](https://github.com/reL4team2/rel4-integral)并将其和[用户态测例](https://github.com/reL4team2/rel4-linux-kit)联合构建运行的脚本。

# usage
仓库下有build-bin.sh和build-lib.sh两个脚本，分别对应使用lib模式和bin模式构建rel4的方式，目前两种方式均可以正常工作。
注：由于内核和用户态测例均在不断更新，您拿到的分发版本可能无法工作，请及时联系我们。
